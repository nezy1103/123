# 🎓 E-Course Platform (MVP)

Система управления электронными курсами — веб-приложение для онлайн-обучения с разделением ролей преподавателей и студентов.

## 📋 Описание

Платформа позволяет преподавателям создавать и управлять курсами, а студентам — просматривать каталог курсов, подписываться на них и изучать материалы.

## ✨ Функциональность

### Для преподавателей:
- ✅ Регистрация и авторизация
- ✅ Создание, редактирование и удаление курсов
- ✅ Просмотр списка своих курсов
- ✅ Отчёт по студентам, подписанным на каждый курс

### Для студентов:
- ✅ Регистрация и авторизация
- ✅ Просмотр каталога всех доступных курсов
- ✅ Подписка на курсы и отписка от них
- ✅ Просмотр своих активных курсов
- ✅ Отчёт: топ курсов по популярности (количеству подписок)

## 🛠 Технологии

- **Backend:** PHP 8.2
- **Database:** MySQL 8.0
- **Server:** Apache 2.4
- **Containerization:** Docker & Docker Compose
- **Architecture:** MVC (Model-View-Controller)
- **Frontend:** HTML5, CSS3 (минимальный стиль)

## 📦 Установка и запуск

### Требования:

- Docker Compose (версия 2.0+)
